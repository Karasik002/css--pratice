В даній роботі розглядається процес розробки веб-сайту, починаючи з вибору структури HTML та закінчуючи впровадженням стилів за допомогою CSS. Особлива увага приділяється використанню сучасних інструментів розробки, таких як Visual Studio Code, що значно спрощують роботу з кодом і дозволяють ефективно керувати репозиторіями через інтеграцію з системою контролю версій Git.

Розробка веб-сайту включає кілька етапів, починаючи з планування структури HTML-документів, які визначають базову архітектуру сайту. Далі йде процес створення та підключення CSS-стилів, що надають сайту привабливий зовнішній вигляд та забезпечують адаптивність для різних пристроїв. Паралельно здійснюється контроль версій коду за допомогою Git, що дозволяє відслідковувати зміни та легко повертатися до попередніх версій проекту.

Під час розробки веб-сайту було успішно реалізовано всі поставлені завдання, включаючи створення ефективної HTML-структури, застосування адаптивних CSS-стилів та інтеграцію з системою контролю версій Git. Використання Visual Studio Code та інших сучасних інструментів значно полегшило процес розробки та дозволило досягти високої якості кінцевого продукту.
